# Loan Repayment Calculator

![UI screenshot]()

## Overview

A demo Android app to calculate loan repayments based on amount, interest rate, and duration.

## Features

- Input fields for loan details.
- Computes monthly payments, total principal, and interest.
- Simple, user-friendly UI.

## Setup

1. **Open the project in Android Studio**:
2. **Run on a device or emulator**.

Project Files

    MainActivity.java: Handles app logic.
    activity_main.xml: Defines UI layout.

Usage

    Enter loan amount, duration, and interest rate.
    Tap "Calculate" to see results.
